# Wallpapers Directory

Place your wallpaper images in this directory.

## Recommended
- `default.jpg` - Default wallpaper for Hyprland (referenced in hyprpaper.conf)
- `lock.jpg` - Wallpaper for lock screen (referenced in hyprlock.conf)

## Supported Formats
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

## Resources
- [Wallhaven](https://wallhaven.cc/)
- [Unsplash](https://unsplash.com/)
- [r/wallpapers](https://www.reddit.com/r/wallpapers/)
- [r/wallpaper](https://www.reddit.com/r/wallpaper/)

## Notes
- Update the paths in `.config/hypr/hyprpaper.conf` and `.config/hypr/hyprlock.conf` if you use different names
- High resolution wallpapers (1920x1080 or higher) work best
